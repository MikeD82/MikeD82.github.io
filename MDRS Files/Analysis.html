Aktualne transkacje z rynku nieruchomości.

Statystyki opisowe

Statystyki opisowe rynku gruntów niezabudowanych. Statystyki dostępne są dla poszczególnych obrębów geodezyjnych.

Aktualnie dostępne statystyki:

  • Średnia,
  • Mediana,
  • Średnia powierzchnia nieruchomości w transakcji,
  • Średnia powierzchnia działki w transakcji,
  • Powierzchnia obrotu,
  • Wartość obrotu,
  • Udział działek o powierzchni < 0.03a,
  • Udział działek o powierzchni 0.03a <= 0.07a,
  • Udział działek o powierzchni 0.07a <= 0.10a,
  • Udział działek o powierzchni 0.10a <= 0.15a,
  • Udział działek o powierzchni 0.15a <= 0.20a,
  • Udział działek o powierzchni 0.20a <= 0.30a,
  • Udział działek o powierzchni > 0.30a,
  • Udział działek o powierzchni 0.07a <= 0.15a,
  • Udział działek o powierzchni 0.07a <= 0.20a,
